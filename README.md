<h2>Web:</h2>

```bash
# Clone este repositório
$ git clone https://github.com/MatheusSESH/GitHub-Explorer.git

# Entre na pasta web
$ cd GitHub-Explorer

# Instale as dependências
$ yarn install

# Execute a aplicação
$ yarn start
```
